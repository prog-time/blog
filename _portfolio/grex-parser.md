## Описание

**Grex Parser** — это парсер маркетплейсов, разработанный на Python, предназначенный для автоматического сбора и анализа данных о товарах. Решение разворачивается на сервере пользователя и позволяет получать структурированную информацию с популярных площадок без использования сторонних SaaS-сервисов.

Основной фокус проекта — автоматизация мониторинга цен, сбора карточек товаров и масштабной выгрузки данных для аналитики, e-commerce и конкурентного анализа. Вся техническая сложность (обход ограничений, обработка страниц, нормализация данных) скрыта за простым веб-интерфейсом.

## Возможности

- Сбор полной информации о товаре по одной ссылке (название, SKU, цены, описание, характеристики, изображения)
- Массовый парсинг категорий, страниц продавцов и списков товаров
- Поддержка маркетплейсов: Wildberries, Ozon, Яндекс Маркет, MegaMarket
- Автоматическое определение площадки по URL
- Экспорт данных в JSON, CSV и XLSX
- Встроенная веб-панель для управления задачами и просмотра результатов
- Ротация прокси (HTTP / SOCKS5) и обход блокировок
- Имитация поведения браузера (User-Agent, cookies)
- Хранение данных исключительно на стороне пользователя (self-hosted)

## Структура системы

- **Parser Core** — модуль сбора и обработки данных с маркетплейсов
- **Jobs Manager** — управление задачами парсинга и очередями
- **Targets** — система управления входными URL (товары, категории)
- **Data Storage** — хранение и структурирование собранных данных
- **Web UI** — интерфейс для запуска задач, просмотра и экспорта данных
- **Proxy Manager** — настройка и автоматическая ротация прокси

## Особенности

- Полный контроль над данными — без передачи в сторонние сервисы
- Отсутствие лимитов на объём парсинга
- Быстрое развёртывание (готовое коробочное решение)
- Не требует технических навыков для ежедневного использования
- Подходит для аналитиков, селлеров и команд e-commerce
