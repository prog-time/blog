## О компании {#about}

One Chat — платформа для управления ботами социальных сетей Telegram и ВКонтакте.

## Задачи {#responsibilities}

Участвовал в разработке платформы для управления ботами социальных сетей Telegram и ВКонтакте.

**Мои обязанности:**
- Разрабатывал кастомные компоненты и модули для Bitrix Framework
- Проектировал и реализовывал REST API для взаимодействия между сервисами
- Разрабатывал интеграции с маркетплейсами (Ozon, Яндекс Маркет)
- Создавал скрипты для синхронизации сайта с CRM-системами Bitrix24 и AmoCRM
- Разрабатывал Telegram-ботов на PHP для автоматизации внутренних и клиентских процессов
- Вёл и поддерживал техническую документацию с использованием Postman и Swagger
