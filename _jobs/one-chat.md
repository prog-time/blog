## О компании {#about}

One Chat — платформа для управления ботами социальных сетей Telegram и ВКонтакте.

## Задачи {#responsibilities}

Участвовал в разработке платформы для управления ботами социальных сетей Telegram и ВКонтакте.

**Мои обязанности:**
- Разрабатывал модули для работы с API популярных соц сетей Telegram и ВКонтакте
- Проектировал и реализовывал REST API для взаимодействия между сервисами
- Обеспечение стабильности и масштабируемости интеграций
- Оптимизация обработки входящих запросов и очередей сообщений

## Технологический стек {#stack}

**Backend:**
* PHP (CodeIgniter 4)
* MySQL

**Frontend:**
* JavaScript

**Интеграции:**
* REST API
* Telegram Bot API
* VK API
* Avito API

**Документация и инструменты:**
* Postman
* Swagger
* Git
