## О компании {#about}

Росбанк — один из крупнейших банков России, входящий в международную финансовую группу.

## Задачи {#responsibilities}

Участвовал в разработке и поддержке микросервисной архитектуры внутреннего продукта. Реализовывал новые бизнес-процессы и интеграции, обеспечивая стабильность и расширяемость системы.

**Мои обязанности:**
- Поддерживал и разрабатывал новые решения для микросервисов на Laravel и Symfony
- Проектировал и реализовывал REST API для микросервисов
- Разрабатывал интеграции для административной панели внутреннего продукта
- Писал unit-тесты для повышения надежности кода
- Использовал инструменты разработки: Jira, GitLab, Postman, Xdebug
